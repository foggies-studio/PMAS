# PMAS — Ultimate Clinical System

**Планирование и анализ лицевой хирургии на базе AI**

> Полнофункциональное веб-приложение для клинического анализа лица, планирования пластических операций и документирования результатов. Работает прямо в браузере — без установки.

---

## ✨ Возможности

### 🧠 AI-анализ лица (FaceMesh)
- Автоматическое распознавание ориентиров лица (TensorFlow.js + MediaPipe)
- Средняя линия, горизонталь глаз, три трети лица
- Индекс гармонии, оценка симметрии, пропорции
- Отклонение носа от средней линии, наклон глаз

### 📐 Измерения и калибровка
- Калибровка по известному расстоянию (мм/пиксель)
- Измерения H и L с автоматическим расчётом AB = √(H² + L²)
- Перетаскивание точек для точной корректировки

### 🔪 Планирование операций
- **Векторы перемещения** — направление и величина подтяжки тканей
- **Наклон (2 точки)** — оценка кантинга, оси носа, линии подбородка
- **Клинический угол (3 точки)** — носогубный угол и др.
- **Зоны** — полигональные области (malar, jowl, neck) с расчётом площади и смещения

### 📋 Готовые клинические пресеты
| Пресет | Описание |
|--------|----------|
| Кантинг глаз | Наклон линии глаз относительно горизонта |
| Ось носа | Наклон оси носа (glabella → tip) |
| Носогубный угол | 3-точечный угол (columella → subnasale → верхняя губа) |
| Линия подбородка | Наклон нижней трети |
| Фейслифтинг: базовые векторы | malar→temporal, jowl→preauricular, mouth→temporal (R/L) |
| Фейслифтинг: SMAS plication | Суперолатеральные векторы + зоны malar/jowl/neck |
| Фейслифтинг: Deep-plane | Глубокие векторы подтяжки + расширенные зоны |
| Фейслифтинг: MACS | Purse-string векторы + зоны |

### ⚖️ Авторасчёт асимметрии
- Автоматическое сравнение R vs L для парных элементов
- Δ длины, Δ угла для векторов и измерений
- Δ площади, Δ смещения для зон

### 📸 До / После
- Сохранение снимка состояния «До»
- Наложение «До» поверх текущего плана для сравнения
- Откат к сохранённому состоянию

### 📄 Экспорт
- **PDF** — полный отчёт с визуализацией
- **DOCX** — структурированный документ для медкарты

## 🚀 Быстрый старт

```
# Клонировать
git clone https://github.com/foggies-studio/PMAS.git
cd PMAS

# Поднять локальный сервер
python3 -m http.server 8080

# Открыть основной интерфейс
open http://localhost:8080/PMAS_Ultimate_Clinical_System_planning_facelift_pro_fixed_plus9.html
```

> **Требования:** современный браузер (Chrome, Edge, Firefox). Для AI-анализа и 3D-страницы нужен интернет для загрузки JS-библиотек.

---

## 🖥️ Как пользоваться

```
1. Загрузите фото пациента (JPG/PNG) или DICOM (.dcm)
2. Выполните калибровку — укажите 2 точки известного расстояния
3. Нажмите «AI ориентиры» для автоматического анализа
4. Выберите пресет или используйте ручные инструменты
5. Экспортируйте результат в PDF или DOCX
```

## 🧊 3D-просмотрщик головы

В PMAS теперь две отдельные страницы: `2D View` для клинической работы и `3D View` для просмотра моделей головы. Между ними можно переключаться через плашки наверху.

```
http://localhost:8080/3d_viewer.html
```

**Функции:**
- Две предустановленные модели: Lee Perry-Smith (детальный скан) и Face Cap (лицо с морфами)
- Загрузка файлов Kiri: `.obj`, `.stl`, `.fbx`, `.gltf`, `.glb`, `.usdz`, `.ply`, `.xyz`
- Режимы: wireframe, нормали, 3 варианта освещения
- Вращение (ЛКМ), перемещение (ПКМ), масштаб (колесо)
- Двойной клик — центрировать модель

### Kiri Engine: как сделать 3D-скан головы

1. В Kiri Engine создайте новый проект через `+` и выберите режим `Photo Scan`.
2. Снимайте голову минимум в 3 поясах: уровень глаз, чуть выше линии волос и нижний пояс вокруг челюсти/шеи.
3. Держите равномерный свет, не размывайте кадры, добавьте крупные планы ушей, висков и линии волос.
4. После обработки экспортируйте модель в один из поддерживаемых форматов Kiri: `OBJ`, `FBX`, `STL`, `GLB`, `GLTF`, `USDZ`, `PLY` или `XYZ`.
5. Откройте страницу `3D View` и загрузите экспортированный файл.

> **Практически:** для Kiri лучше использовать `GLB`. Один файл проще переносить в PMAS, и он надёжнее загружается локально.
